Charts

A chart (graph) is a pictorial representation of statistical data used to make it easier to understand and interpret information.

Velocity Chart

The velocity chart in the Board page analyses and displays the story points in the beginning, in the middle and at the end of the sprint. We can analyse the tasks and the teamâ€™s consistency in completing them using the velocity chart.

Team Performance Chart

The team performance chart analyses and displays the performance of the entire team over a range of sprints. We can also check the individual performance of team members using the filter to select an individual.

Sprint Burndown Chart

The sprint burndown chart analyses and displays the number of days the team members took in a sprint to complete the tasks. It compares the story points in tasks with the number of days taken to finish that task.
