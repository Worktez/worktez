My Organisation

The “My Organisation” page displays all the teams in an organisation along with details such as members in the team, total team tasks, manager email, current sprint and GitHub repository or handle if linked.

  1. The manager can integrate their GitHub repository using the “Integrate GitHub” button.
  2. The “Add Member” button displays a popup when clicked, which will allow us to add a new member to the team.
  3. The “Trash Bin” icon on the top right corner of the team card allows us to delete the team.
  4. The Gear icon when clicked redirects us to a page with team details, where we can add or remove the members and also customise the team labels.
  5. All these operations on the teams can be performed only by the admins of the team.

Filters

The Filters page allows the user to create a custom filter to sort data and get specific options according to the details specified such as difficulty, priority, status, sprint number, assignee, and team. The created filters will be saved in the filter page for future use.

Multiple Organisations

The header contains the organisation name on the left beside the organisation icon. It contains a dropdown to select the organisations to display the sprint details in it. This feature is accessible to users who are enrolled in multiple organisations.
